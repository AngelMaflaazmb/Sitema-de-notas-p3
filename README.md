
# Sistema de Notas
Angel Zahid Mafla Bolaños

**Objetivo**

Desarrollar un sistema en Python que permita registrar las calificaciones de varios estudiantes, calcular su promedio y determinar automáticamente si aprueban o reprueban la asignatura.

**Descripción del proyecto**

El programa muestra una lista de estudiantes, permite ingresar cinco calificaciones para cada uno, verifica que las notas estén entre 0 y 10, calcula el promedio y muestra el estado académico correspondiente.

El sistema también permite repetir el proceso para ingresar nuevas calificaciones cuando el usuario lo desee.

**Funcionalidades**

Muestra una lista de estudiantes.

Permite ingresar cinco notas por cada estudiante.

Valida que las calificaciones estén entre 0 y 10.

Calcula el promedio de las notas ingresadas.

Indica si el estudiante está APROBADO o REPROBADO.

Permite repetir el proceso de ingreso de notas.

Utiliza un diccionario para almacenar la información de los estudiantes.

**Estructuras utilizadas**

Diccionario **(dict)** para almacenar la lista de estudiantes.

Lista **(list)** para guardar las calificaciones.

Condicionales **(if y else).**

Bucles **(for y while).**

**Tecnologías** utilizadas
Python 3
